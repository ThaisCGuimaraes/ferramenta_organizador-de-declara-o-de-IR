# ferramenta_organizador-de-declara-o-de-IR
Ferramenta desenvolvida no Excel que ajuda a organizar e reunir informações essenciais para a declaração de imposto de renda. 
O arquivo possui 3 telas para preenchimento de forma configurada para gerar um banco de dados limpo e funcional. As telas ção configuradas com botoes de transição intuitivas.
